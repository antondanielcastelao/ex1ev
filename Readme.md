# Juego de las damas
## Funcionamiento
El juego consta de un **tablero de 8 x 8**. 
Las fichas blancas son un círculo ('O') y las negras
se representan con una cruz ('X'). __Va ganando el juegador 
que más fichas tenga en el tablero__.
